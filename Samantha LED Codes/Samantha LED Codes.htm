Performed when power is applied to the Samantha Module

If all is well, Red LED will remain on after power on reset seq.

White LED will blink if there is a Network Found that matches one in Network Configurations on Samantha

Blue LED will remain on when NXT is powered on

1st time Samantha is connected to FCS w/ password. MUST PRESS RED BUTTON TO ACCEPT PASSWORD

After password is accepted, Blue LED will pulse, indicating data flow.

NXT not found, either unplugged or not powered on.

Found NXT, no network has been found. Reconfigure Network on Samantha w/ USB

Found network, but Samantha battery level <10 Vdc

Found network and NXT but Samantha battery level <10Vdc. If connected to FCS, only Red LED will blink

NXT has either been disconnected, lost power, or powering down.

 

Samantha Setup Instructions

Configuring Network

Firmware Loading

1.       Load <4g USB Drive with file from Samantha Network Config program (AT COMPETITIONS, THIS WILL BE PROVIDED)

2.       Start with Samantha powered OFF

3.       Plug in USB Drive

4.       POWER ON Samantha Module

5.       Wait for Power-on LED Sequence (above) to occur TWICE

6.       POWER OFF Samantha Module

7.       Unplug USB Drive

8.       Plug in NXT Cable

9.       Power ON- WHITE LED should flash

1.       Load <4gb USB Drive with Samantha.hex file (usually found in C:\Program Files\Samantha Field Control Systems\Samantha on Windows)

2.       Start with Samantha powered OFF

3.       Plug in USB drive

4.       Hold down RED button on Samantha

5.       POWER ON Samantha Module

6.       Wait to see above LED Sequence (Starts when RED blinks twice, ends with Sequence) TYPICALLY TAKES 20-30 Seconds

7.       POWER OFF Samantha Module

8.       Remove USB drive

9.       Plug in NXT Cable
